
# Classification of Phishing Website

A Machine Learning based web application that detects phishing websites using URL-based features.

## Features
- Real-time phishing website prediction
- Trained on 42K+ URLs
- 87% model accuracy
- Flask-based web interface
- Responsive frontend UI

## Tech Stack
- Python
- Scikit-learn
- Flask
- HTML
- CSS

## Run Project

```bash
pip install -r requirements.txt
python app.py
```
