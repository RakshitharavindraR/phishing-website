# Classification of Phishing Website

A Machine Learning based web application that detects phishing websites using URL-based features and predicts whether a website is safe or phishing.

## Features
- Real-time phishing website prediction
- Trained on 42K+ URLs
- 87% model accuracy
- Flask-based web application
- Responsive user interface
- URL feature extraction

## Tech Stack

### Frontend

- CSS3

### Backend
- Python
- Flask

### Machine Learning
- Scikit-learn
- Pandas
- NumPy
